# ecommerce-customer-analytics
Customer segmentation, RFM analysis, and sales forecasting on real e-commerce data
## Dataset
Download the Olist Brazilian E-Commerce dataset from:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
## Dashboard
View the live interactive dashboard:
[E-Commerce Customer Analytics](https://public.tableau.com/app/profile/touzani.omar/viz/dashboard_ecommerce_customer_analytics/Revenue_overview)
## Project Phases
- Phase 1: Exploratory Data Analysis
- Phase 2: Data Cleaning
- Phase 3: RFM Analysis
- Phase 4: K-Means Clustering
- Phase 5: Sales Forecasting
- Phase 6: Tableau Dashboard

## Dataset
Download from Kaggle:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
Place all CSV files in data/raw/ before running notebooks.
