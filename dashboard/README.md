# E-Commerce Customer Analytics — Tableau Dashboard

## Live Dashboard
View the interactive dashboard on Tableau Public:
[E-Commerce Customer Analytics Dashboard](YOUR_TABLEAU_PUBLIC_URL_HERE)

## Dashboard Pages

### Page 1 — Revenue Overview
- KPI tiles: Total Revenue, Total Orders, Avg Order Value, Avg Review Score
- Monthly Revenue Trend with average line and trend line
- Revenue by State (Brazil map)
- Top 10 Product Categories
- Review Score Distribution

### Page 2 — Customer Segments
- RFM Segment Distribution
- Revenue by Segment with percentage labels
- Cluster Profiles (K-Means)
- RFM Scatter Plot — Recency vs Spend

### Page 3 — Sales Forecast
- Revenue Historical & Forecast
- Model Comparison — MAPE
- Forecasting Methodology

## Data Sources

- revenue_combined.csv
- rfm_scores.csv
- rfm_segment_profiles.csv
- cluster_profiles.csv
- model_comparison.csv
